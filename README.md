# V.E.W Contracting Database System

Relational database project developed in MySQL for managing construction business operations.

## Overview

This database system was designed for V.E.W Contracting to organize and manage customers, projects, employees, materials, and invoices.

## Technologies Used

- MySQL
- MySQL Workbench
- SQL

## Features

- EER diagram
- Relational database structure
- Primary and foreign key relationships
- Customer, project, employee, material, and invoice tables
- SQL implementation for business data management

- EER Diagram.mwb
- SQL scripts
